# INI-CET 28-Day Study Planner

A focused 28-day war plan for the INI-CET / NEET PG exam (May 16, 2026).

Built with React + Vite. All 28 days of study data embedded — no backend needed.

## Features
- Live countdown to exam day
- 28-day study grid with phase color-coding (Blitz / Rapid Revision / Mock)
- Subject filters
- Detailed day cards: Topics, India-specific content, Images, MCQ targets, Notes
- Daily schedule view
- Personal notes — persisted in localStorage

## Stack
- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- lucide-react icons
- pnpm monorepo
